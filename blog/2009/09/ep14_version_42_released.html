EP1-4 version 4.2 Released

Update patch finished for the 4 question arcs.

An installer has been added for the Mac OSX version with many thanks to Roto, the newest member of our team.

New features for all versions include a ctrl skip that is much faster (the 's' skip is the same speed that it used to be), and song titles and artist names have been translated/romanized as the case may be.  The titles used at the moment are temporary; we hope to ask the artists themselves if they have any preferences and include those in the next update.  Also, many of the larger picture files have been converted into png format to save space.

Though many edits have been made to all parts of Episode 1, chapters 14 and onwards have been sifted through line by line.  Total number of lines that have been changed for accuracy or flow is more than 1300...yeah, seriously.

This version of the patch does not include a full set of edits based on new info from EP5.  Hopefully, a new version will be released along with the EP5 patch when that is finished.
